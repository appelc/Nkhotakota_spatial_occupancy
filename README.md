# Nkhotakota_spatial_occupancy

Code for spatial occupancy models with camera trap data from Nkhotakota Wildlife Reserve

Example data are included in the `data` folder

The code is separated into 3 steps:

### PREP:

preparing data, covariates, and inputs for spOccupancy package

### RUN:

run the spatial occupancy model(s)

### REPORT:

conduct posterior predictive checks, derive parameter estimates, conduct trend analysis, and create figures
